Program Description: 
This program is an entry level tool for CS:GO beginners. It allows you to select multiple weapons, compare their stats, and find out 
(through a bubble sort algorithm) which weapon has the best stats out of the group of guns. In addition, there will be external files
for each gun’s stats that could be edited and altered along with CS:GO updates. 

Instructions:
1. Launch program, press F6 (or the green play button up top).
2. Select desired weapons through combo boxes.
3. Press "Compare Items" button at the bottom of the program.
